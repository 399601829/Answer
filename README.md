# Answer

> Answer-基于领域本体的问答系统服务端

## 调试开发
* 下载项目源码
* 使用 Eclipse、Idea 或其它 IDE 导入项目并设置好 maven、jdk 等配置
* 将 libs 下的 jar 包添加成项目 Library
* 运行 cn.lcy.answer.AnswerMain
* 通过浏览器访问 http://localhost:8080/answer?q=“问题”
## 调用 API
```
GET localhost:8080/answer?q=xxx
```
